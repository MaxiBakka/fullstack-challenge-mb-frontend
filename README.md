
## Description

React + TypeScript + Vite

## Development


1. Go to folder, and copy `env-example` as `.env`.

   ```bash
   cd my-app/
   cp env-example .env
   ```

1. Then, install all the needed dependencies using your desired package manager,

   ```bash
   npm i
   ```

2. Serve with hot reload at http://localhost:5173.
```bash
   npm run dev
   ```
